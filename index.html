📈 Some details about this graph

- Last measurement taken ~ seconds ago
- Shows the last 24hrs of data (currently showing ~ points)
- Null values are skipped
- The values are actually averages of 15(ish) measurements

💻 Tech behind it

- Arduino Esp8266 (low-cost Wi-Fi microchip)
- DHT22 temp sensor
- thingspeak (IoT data logger)
- Netlify (hosting static site)
- ChartJS (graphing)
